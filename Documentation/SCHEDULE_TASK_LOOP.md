В папке `Программа` могут временно появляться служебные lock-файлы Microsoft Word, например:

`Программа/~$_Ознакомительная_Бартов_2026_Зима.docx`

Это не рабочие документы. Они создаются при открытии DOCX на стороне пользователя и обычно удаляются автоматически после закрытия Word.

======================================================================
                      DATE CALCULATION RULES
======================================================================

При расчёте нового графика использовать производственный календарь РФ на соответствующий год.

Общие правила:

1. Занятия идут только по указанным дням недели.
2. Количество занятий сохраняется.
3. Если дата занятия попадает на официальный нерабочий праздничный день РФ — эту дату пропустить.
4. После пропуска продолжить график по следующему подходящему учебному дню.
5. Если рабочий день перенесён на субботу, субботу не добавлять автоматически, если группа не занимается по субботам.
6. Не додумывать новые дни занятий без указания пользователя.
7. Если есть сомнение по праздничному дню или переносу — записать это в `QUESTIONS`, а не менять график наугад.

Для последующих итераций пользователь может давать задачу в двух сообщениях.

2. Прочитать `БЛОК 1. CURRENT TASK`.
3. Прочитать `БЛОК 2. PREVIEW / SHORT COMMAND`.

5. Прочитать `DOCUMENT STRUCTURE NOTES`.

9. Проверить результат по разделу `REVIEW`.
10. После выполнения обновить `AGENT REPORT`.

Важное правило:

Разовая задача должна находиться только в `БЛОК 1. CURRENT TASK`.
